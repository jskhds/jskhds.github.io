4.this

this 在函数执行的时候才确定值，不是在函数定义的时候确认的

4.1调用场景

  • 作为普通函数 返回window

  • 使用 call apply bind

    • ```js
      function fn1(){
      console.log(this);
      }
      fn1() // window
      // call 和 bind 可以改变函数 this 的指向 但是用法有所不同
      fn1.call( {x:100} ) // { x: 100 }
      // bind 返回一个新的函数执行
      const fn2 = fn1.bind( {x:200} )
      fn2() // { x: 200 }

  • 箭头函数 setTimeout里是箭头函数 指向上一级this

    • ```js
      const zhangsan = {
       waitAgain(){
     setTimeout(()=>{
         console.log(this);
     })
 }
      }
      zhangsan.waitAgain() // 当前对象 也就是 zhangsan 这个对象
