2. 模板编译

我们都知道，html 是标签语言，不像 JS 一样，可以实现判断、循环。而我们的模板里面，既有标签，也有指令和插值，还有 JS 表达式，所以 模板 肯定是转换为某种 JS 代码，即编译模板。

这个转换的过程主要可以分为三步

  • 模板编译为 render 函数，执行 render 函数返回 vnode
  • 基于 vnode 再执行 patch 和 diff
  • 使用 webpack vue-loader 在开发环境下编译模板
