1） Map 和 Set 有序和无序
  • 有序慢，无序快。但是无序本身就是一个缺点。比如说 v-dom里面，object什么的可以无序排列，但是数组必须是有序的。不然渲染过程会出现问题
  • 可以结合二者优点：二叉树及其变种
2）Map 和 Object 区别
  • API 不同（略过，比较简单），Map可以以任意类型为 key
  • Map 是有序结构（重要）
  • Map 操作同样很快 对比无序结构object 的 查找和删除操作 map 甚至会更快一点。
